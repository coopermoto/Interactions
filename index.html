Interactions by JoeKunk

A database application in Microsoft.Net for managing community service interactions

Welcome to the Interactions project. This project is based upon the ON VB article that appeared in the March 2012 issue of Visual Studio Magazine on database synchronization. All interested developers are welcome to join this project and contribute code to assist charitable organization track interactions with clients that may need community services to support their quality of life.
